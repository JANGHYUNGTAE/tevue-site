듀얼 스크린 기기 연결 가이드

차량 화면에 폰과 다른 앱(예: 내비)을 띄우는 듀얼 스크린 최초 설정 방법입니다.

📱 Android 전용 기능입니다(iPhone 미지원). 보통 한 번 설정해두면 되고(재부팅 시 재연결이 필요할 수 있어요). 무선 디버깅을 이용하며, 별도 PC나 앱 설치는 필요 없습니다.
📶 핫스팟 대역을 5GHz로 설정하면 더 원활하게 미러링/듀얼 스크린이 동작해요. (설정 › 핫스팟 › 대역 → 5GHz)

개발자 옵션 켜기

휴대전화 설정에서 개발자 옵션을 활성화합니다.

설정 › 휴대전화 정보 › 소프트웨어 정보 › 빌드번호 7번 연속 탭

“개발자가 되었습니다” 문구가 뜨면 성공입니다.

무선 디버깅 켜기

방금 생긴 개발자 옵션에서 무선 디버깅을 켭니다.

설정 › 개발자 옵션 › 무선 디버깅 ON

TeVue에서 ‘자동 페어링 시작’ 누르기

TeVue 앱 › 기기 연결 화면에서 파란색 ‘자동 페어링 시작’ 버튼을 누릅니다.

‘페어링 코드로 기기 페어링’ 선택

무선 디버깅 화면에서 ‘페어링 코드로 기기 페어링’을 누르면 6자리 코드가 표시됩니다.

개발자 옵션 › 무선 디버깅 › 페어링 코드로 기기 페어링

6자리 코드 입력 → 연결 완료

TeVue 알림(또는 입력창)에 그 6자리 코드를 입력하면 자동으로 연결됩니다. “준비 완료” 상태가 되면 끝입니다.

듀얼 스크린 시작

이제 ‘듀얼 스크린 시작’으로 차량 화면에 원하는 앱을 띄울 수 있습니다.

⚠️ 폰을 재부팅하면 무선 디버깅이 꺼질 수 있습니다. 그럴 땐 무선 디버깅을 다시 켜고 ‘자동 페어링 시작’으로 다시 연결하면 됩니다. (Android 보안 정책상 정상 동작입니다.)
